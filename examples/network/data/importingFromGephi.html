Dynamic Data - Importing from Gephi (JSON)

This example shows how to import a JSON file exported by Gephi. The two options available for the import are available through the checkboxes. You can download the Gephi JSON exporter here: https://gephi.org/plugins/#/plugin/jsonexporter-plugin. All of Gephi's attributes are also contained within the node elements. This means you can access all of this data through the DataSet.
