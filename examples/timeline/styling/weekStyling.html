Week numbers are calculated based on locales. For this to properly work, the timeline must be loaded with locales. To localize the Timeline, one has to load a version of moment.js including locales. To set a locale, specify option {locale: STRING}. To set the scale to use week numbers, use for example {scale: 'week', step: 1}. For example, the 'en' locale supports week number calculations using US week numbers. Check out what's happening if you select another locale.
